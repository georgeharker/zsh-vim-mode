# zsh-vim-mode

Vi editing for your zsh command line with prompt redraws that actually work:
the mode indicator and cursor shape always reflect the current mode, the
plugin composes with oh-my-posh, zsh-autosuggestions, and friends regardless
of load order, and everything is configured with `zstyle`.

Everything — install, configuration, prompt integration, key bindings — lives
in the [README](README.md).
